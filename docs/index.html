Installation

Follow these steps to install the Hero Slider plugin:

  1. Download the plugin from the official repository.
  2. Upload the plugin folder to the /wp-content/plugins/ directory.
  3. Activate the plugin through the **Plugins** menu in WordPress.
  4. Go to **Settings > Hero Slider Settings** to configure your slides.

Usage

Learn how to use the Hero Slider plugin and configure your slides:

  • Go to **Settings > Hero Slider Settings**.
  • Enter the image URL, heading, paragraph, and button details for each slide.
  • Use the shortcode [hero-slider] on any page or post to display your slider.

Troubleshooting

If you're having issues with the Hero Slider plugin, here are some common problems and solutions:

  • Slider not appearing: Make sure you have added slides in the **Settings > Hero Slider Settings** page.
  • Images not loading: Ensure the image URLs are correct and publicly accessible.
  • Buttons not working: Double-check that the button link is correct and leads to a valid URL.
  • Mobile responsiveness issues: Try adding custom CSS to adjust the slider on mobile devices.
